1 三阶段动画分镜

每阶段为可独立讲解的视觉单元，阶段间用「逐根出现」过渡动画衔接。下方为 Three.js 实时 3D 预览，可旋转观察。

阶段 0 · 空坐标系0 / 2
状态：仅渲染地面网格 + 三轴（X/Y/Z），标注坐标。
目的：建立空间参照，让学生预期后续构件落点。
阶段 1 · 蓝立柱升起1 / 2
状态：沿椭圆环逐根升起 蓝色立柱（N=10）。柱顶高度 = 双曲抛物面在该(x,y)处的 z 值。
过渡：逐根从 z=0 升至目标高度。

2 核心数学：双重直纹面证明

曲面方程与因式分解

% 标准双曲抛物面（马鞍面） z = x²/a² − y²/b² % 因式分解 → 关键！ z = (x/a − y/b)(x/a + y/b) % 令 u = x/a − y/b , v = x/a + y/b z = u · v

固定 u=u₀ ⟹ z 随 v 线性变化 ⟹ 一条直线。
固定 v=v₀ ⟹ z 随 u 线性变化 ⟹ 另一条直线。
过曲面每点恰有两条母线 → 双重直纹面。

每组母线用「两端点」定义（证明笔直）

% u=const 这组，取两个不同的 v： P₁ = ( x(u₀,v₁), y(u₀,v₁), z(u₀,v₁) ) P₂ = ( x(u₀,v₂), y(u₀,v₂), z(u₀,v₂) ) % 反解：x = a(u+v)/2, y = b(v−u)/2, z = u·v L(t) = P₁ + t·(P₂ − P₁), t∈[0,1] % 母线 = P₁ 到 P₂ 的线段 → MATLAB plot3 一笔直线

每条母线只取两个端点 plot3 即可，无需采样中间点。 单元测试：线段中点 z 值应等于曲面 z 值（容差 1e-10）→ 证明是直线。
