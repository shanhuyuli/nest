## Problem Statement

当前项目（迭代1）的鸟巢动画与真实鸟巢形状无关：只有一个黑色圆环、简单马鞍面和稀疏线条。需要修复为基于真实鸟巢双曲抛物面构造原理的多阶段课堂演示动画，并用 MATLAB 和 Python 双语言独立实现。

核心教学目的：**演示双曲抛物面是直纹面——可通过两族直线交叉编织生成**。

---

## Solution

基于国家体育场（鸟巢）的实际构造原理，重构动画：

1. **内外椭圆结构**：外椭圆代表体育场外壳（332m×296m 实际比例缩放），内椭圆代表屋顶开口
2. **直母线编织屋顶**：利用双曲抛物面（Hyperbolic Paraboloid）的双重直纹特性，通过 u 族和 v 族直母线与内外椭圆求交，生成规则交叉编织网
3. **三阶段教学动画**：空坐标系 → 立柱逐根生长 → 屋顶线逐条铺出
4. **纯直线构造**：不渲染半透明曲面和底面圆圈，仅用直线表现鸟巢结构

## Implementation Decisions

### 1. 数学几何模型

**双曲抛物面方程**（含垂直偏移）：
```
z = c · (x²/ahp² − y²/bhp²) + z_offset
```

**核心参数**（原型验证通过）：

| 参数 | 值 | 说明 |
|------|-----|------|
| ahp | 80 | HP 曲面 a 参数（独立于椭圆） |
| bhp | 72 | HP 曲面 b 参数 |
| c | 13.5 | 翘起量（控制马鞍起伏） |
| z_offset | 23.5 | 垂直偏移（确保所有立柱 ≥ 16） |
| a_out, b_out | 60, 53.5 | 外椭圆（外壳投影） |
| a_in, b_in | 34.4, 22.4 | 内椭圆（开口投影） |

**直母线族**：
- u 族（斜率 +bhp/ahp）：x/ahp − y/bhp = u，u ∈ [-1.06, 1.06]，50 条均匀分布
- v 族（斜率 −bhp/ahp）：x/ahp + y/bhp = v，v ∈ [-1.06, 1.06]，50 条均匀分布

**线段连接规则**：
```
对每条直母线 L：
  1. L 与外椭圆求交 → 0~2 个交点
  2. L 与内椭圆求交 → 0~2 个交点
  3. 所有交点按直线参数排序
  4. 标记每个点为「外」或「内」
  5. 连接相邻交点：
     - 跳过 内→内（在开口区域内）
     - 跳过线段中点在内椭圆内的连接
     - 保留 外→外、外→内、内→外
  6. 每段线段两端点各放一根立柱（z=0 → z=hpZ(x,y)）
```

### 2. 动画分镜

| 阶段 | 内容 | 时长 | 过渡 |
|------|------|------|------|
| Stage 0 | 坐标轴 + 地面网格 | ~1s | — |
| Transition 0→1 | 立柱逐根生长 | ~2-3s | 先外圈后内圈，每帧新增若干根立柱 |
| Stage 1 | 所有立柱完成 | ~2s | — |
| Transition 1→2 | 屋顶线逐条铺出 | ~3-4s | 先 u 族再 v 族，每条从一端延伸 |
| Stage 2 | 完整编织网 | ~3s | — |

### 3. 视觉规范

| 元素 | 颜色 | 线宽 | 说明 |
|------|------|------|------|
| 立柱 | 浅灰 [0.6, 0.6, 0.6] | 3pt | 仿钢结构色 |
| 屋顶线 | 橙色 #EDB120 | 2pt | MATLAB 标准橙 |
| 坐标轴 | 深灰 #333333 | default | — |
| 背景 | 白色 | — | 投影仪友好 |

### 4. 相机与渲染

- 视角：azimuth = -35°, elevation = 30°（侧俯视，全程固定）
- 图窗大小：800×600
- 数据纵横比：daspect([1 1 1])
- Z 轴范围：zlim([0, max_z])
- 帧率：15-20 fps
- 导出格式：mp4 (VideoWriter) + gif (exportgraphics)

### 5. 代码架构

**模块化三文件结构**（MATLAB 和 Python 各自独立实现相同架构）：

- **Geometry Module**：纯函数，参数 → 几何数据（立柱坐标、屋顶线段端点）
- **Animation Module**：接受几何数据，操作 figure/axes 渲染逐帧动画
- **Driver**：参数配置 → 调用 Geometry → 调用 Animation → 可选导出

### 6. 立柱与母线的空间关系

- 立柱位置 = 直母线与椭圆的交点（方式 B），保证结构逻辑一致性
- 不同直母线可能在相近位置产生交点 → 不去重，体现节点汇交特征
- 内圈立柱也从 z=0 升起（与外圈统一），教学逻辑清晰

### 7. 原型验证结论

通过 MATLAB 原型（96 条母线）验证：
- 立柱顶端精确落在 HP 曲面上（误差 = 0）
- 直母线中点在 HP 曲面上（误差 ≈ 1e-15，证明直纹面特性）
- 所有立柱高度 ≥ 16（明显高于地面）
- 线段连接算法正确排除开口区域

## Testing Decisions

### 测试接口（Seams）

- **Seam 1**: `nest_geometry(params) → geometry_data` — 纯函数，可独立单元测试
- **Seam 2**: `nest_animation(geometry_data, params, ax) → void` — 操作图形，测试帧数据
- **Seam 3**: `run_nest_demo(params) → file` — 集成测试，检查输出文件

### 单元测试

- **MATLAB**：`test_nest_geometry.m`
  - 验证 hyperboloidZ 函数返回值正确
  - 验证 lineEllipseIntersect 交点计算正确
  - 验证直母线中点落在 HP 曲面上（容差 < 1e-10）

- **Python**：`test_nest_geometry.py`（pytest）
  - 与 MATLAB 相同逻辑的三项测试

### 测试原则

- 只测试外部行为（输入→输出），不测试实现细节
- 优先使用现有测试框架（MATLAB 脚本测试、pytest）
- 纯函数（geometry）全面测试；渲染函数（animation）轻量验证

## Out of Scope

- **不**追求真实鸟巢的复杂次结构、编织纹理或材料渲染
- **不**做交互式 GUI / App Designer
- **不**支持多语言字幕、配音、配乐
- **不**支持实时参数滑块界面
- **不**使用真实鸟巢的平移曲面方程（R₁沿R₂扫掠），仅使用 HP 近似
- **不**合并去重相近位置的立柱
