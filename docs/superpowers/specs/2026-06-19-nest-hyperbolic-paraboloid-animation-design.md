## 1. 背景与目的

课堂需要直观演示一个核心数学事实：**双曲抛物面（马鞍面）是双重直纹面** —— 它可以完全由直线构成，且过曲面上每一点恰好有两条直线落在曲面上。

以「鸟巢」（国家体育场）的真实结构为载体：外围一圈立柱支撑一个马鞍形屋顶，而屋顶的钢骨架就是双曲抛物面的两组直纹母线。通过动画分阶段「建起」这个结构，让学生亲眼看到：

1. 两组母线**确实是直线**（一条母线只需两个端点定义，中间不采样即落在曲面上）；
2. 这两组直线交织，**勾勒出整个马鞍面**；
3. 立柱顶点**贴合**曲面的高低起伏。

## 2. 目标与非目标

### 目标
- 交付一段可在课堂 MATLAB 中实时运行的多阶段动画。
- 同时提供导出 `mp4`（必要时附 `gif`）的视频版本，脱离 MATLAB 可播放。
- 动画清晰呈现三阶段：空坐标系 → 蓝立柱 → 橙母线屋顶。
- 阶段间用「逐根出现」过渡，节奏适合讲解。
- 视角为侧面俯视（az=−35°, el=30°），马鞍两向翘起都可见。
- 代码模块化，几何部分可独立单元测试（验证母线是直线）。

### 非目标 (YAGNI)
- **不**追求真实鸟巢的复杂次结构/编织纹理/材料渲染（仅取其「椭圆环立柱 + 马鞍屋顶」的几何骨架）。
- **不**做交互式 GUI / App Designer（纯脚本驱动即可）。
- **不**支持多语言字幕、配音、配乐。
- **不**支持实时参数滑块界面（参数集中在脚本顶部常量，改完重跑）。

## 4. 核心数学

### 4.1 曲面方程

标准双曲抛物面（马鞍面）：

```
z = c · (x²/a² − y²/b²)
```

其中 `a, b` 取椭圆长短半轴，`c` 为翘起缩放量（控制马鞍起伏明显度，默认 2.5）。

### 4.2 因式分解 → 双重直纹证明

```
z = c · (x/a − y/b)(x/a + y/b)
```

引入参数 `u, v`：

```
u = (x/a − y/b) / 2
v = (x/a + y/b) / 2
```

则 `x = a(u + v)`, `y = b(v − u)`, 代入得 `x²/a² − y²/b² = (u+v)² − (v−u)² = 4uv`，因此 `z = c·(x²/a² − y²/b²) = 4c·u·v`。

> 实现时统一采用 §4.3 的 `/2` 标度（`x=a(u+v)/2`），则 `x²/a² − y²/b² = uv`，`z = c·u·v`。两者只差一个 u,v 的标度常数（§4.2 的 u 是 §4.3 的 2 倍），曲面形状完全相同。下文一律以 §4.3 标度为准。

**证明**：固定 `u = u₀`，则 `z = c·u₀·v` 随 `v` **线性**变化 → 落在曲面上的是一条**直线**。同理固定 `v = v₀` 得另一条直线。因此过曲面上每点 `(u₀, v₀)` 恰有两条母线，曲面是双重直纹面。

### 4.3 参数化（实现用）

令 `u, v ∈ [−1, 1]`（规范化域），映射：

```
x = a · (u + v) / 2
y = b · (v − u) / 2
z = c · u · v
```

> 校验：当 `u=v=1` → x=a, y=0, z=c；当 `u=1,v=−1` → x=0, y=−b, z=−c。符合马鞍「长轴两端上翘、短轴两端下垂」的几何。

### 4.4 母线的「两端点」定义（关键）

每条母线只需取两个端点，`plot3` 一笔画直线，**无需中间采样**：

- **u = const 组**：取 `v₁ = −1` 与 `v₂ = 1`：
  - `P₁ = (x(u,v₁), y(u,v₁), z(u,v₁))`
  - `P₂ = (x(u,v₂), y(u,v₂), z(u,v₂))`
- **v = const 组**：取 `u₁ = −1` 与 `u₂ = 1`：
  - `P₁ = (x(u₁,v), y(u₁,v), z(u₁,v))`
  - `P₂ = (x(u₂,v), y(u₂,v), z(u₂,v))`

母线条数取 `k = 1 … M−1` 处的 `u = −1 + 2k/M`（跳过边界 `u=±1`，因其重合于椭圆边界，避免冗余）。默认 `M_u = M_v = 9` → 每组 8 条内母线。

### 4.5 立柱位置与高度

立柱沿椭圆环均匀分布：

```
θ_i = 2π · i / N,   i = 0 … N−1
x_i = a · cos(θ_i)
y_i = b · sin(θ_i)
柱顶 z_i = c · (x_i²/a² − y_i²/b²)
```

> 注：`(cos θ, sin θ)` 使 `(x/a)²+(y/b)²=1`，故立柱恰在椭圆边界上。柱顶 `z_i = c·(cos²θ_i − sin²θ_i) = c·cos(2θ_i)`，自然产生长短轴方向的高低差（长轴端 z=+c 高翘、短轴端 z=−c 下沉），呈现马鞍。

## 6. 代码架构（模块化）

```
src/matlab/
├── nest_geometry.m       % 纯几何：返回立柱坐标、母线端点、曲面网格
├── nest_animation.m      % 3 阶段渲染 + 逐根过渡（操作 figure/axes/plot3）
├── run_nest_demo.m       % 驱动入口 + record 开关 → VideoWriter 导出
└── test/
    └── test_nest_geometry.m  % 单元测试（验证母线是直线）
```

### 6.1 `nest_geometry.m`

纯函数，无副作用，可独立测试。

**输入**：参数结构体 `p`（含 a, b, c, N, M_u, M_v）。

**输出**：结构体 `G`：
- `G.pillars`：`[3×N]` 矩阵，每列一根立柱的 (x, y, z_top)。
- `G.rule_u`：cell `{1×(M_u−1)}`，每个 cell 是 `[3×2]` 的两端点（u=const 母线）。
- `G.rule_v`：cell `{1×(M_v−1)}`，同上（v=const 母线）。
- `G.surf_mesh`：`[X, Y, Z]` 网格（供 `surf` 半透明片）。
- `G.ellipse_pts`：`[2×128]` 椭圆边界点（供画地面环）。

### 6.2 `nest_animation.m`

**输入**：`G`（几何）、`p`（参数）、`ax`（axes 句柄，可选）、`recorder`（VideoWriter 句柄，可选）。

**行为**：分三阶段渲染，每阶段内逐根出现：

1. **阶段 0 · 空坐标系**：`hold on`，画椭圆地面环 + 三轴标注（X/Y/Z），`view([az,el])`，`axis equal`，`pause(hold_t)`。
2. **阶段 1 · 蓝立柱**：`for i=1:N`，`plot3([x_i,x_i],[y_i,y_i],[0,z_i],'Color',#0072BD,'LineWidth',2)`，每根后 `drawnow; pause(dt)`；若 `recorder` 存在则 `writeVideo` 逐帧。
3. **阶段 2 · 橙母线**：先画半透明 `surf(X,Y,Z,'FaceAlpha',0.2,'EdgeColor','none')`；再逐条 `plot3` 两组母线（`#EDB120`），每条后 `drawnow; pause(dt)`。

> 「逐根」实现：用循环 + `drawnow`/`pause`；录制模式下每根追加数帧（如 `round(dt*fps)` 帧）到 VideoWriter。

### 6.3 `run_nest_demo.m`

驱动入口：

```matlab
function run_nest_demo(opts)
% opts.record (false|true), opts.out ('nest.mp4'), opts.gif (false|true)
  p = default_params();              % §5 默认值
  G = nest_geometry(p);
  fig = figure('Color','w','Position',[100 100 900 700]);
  ax  = axes(fig);
  if opts.record
    vw = VideoWriter(opts.out, 'MPEG-4'); vw.FrameRate = p.fps; open(vw);
    nest_animation(G, p, ax, vw);
    close(vw);
    if opts.gif, nest_to_gif(opts.out, [opts.out '.gif']); end
  else
    nest_animation(G, p, ax, []);
  end
end
```

### 6.4 `test_nest_geometry.m`

单元测试，验证几何正确性：

1. **母线中点在曲面上**：对每条母线两端点 `P₁, P₂`，取中点 `M = (P₁+P₂)/2`，验证 `M.z ≈ c·(M.x²/a² − M.y²/b²)`，容差 `1e-10`。→ 证明线段在曲面上是直线。
2. **立柱在椭圆上**：每根立柱 `(x_i, y_i)` 满足 `(x_i/a)²+(y_i/b)² = 1`。
3. **柱顶贴合曲面**：`z_i ≈ c·(x_i²/a² − y_i²/b²)`。
4. **u=const 母线为直线**：除中点外，任取 `t∈[0,1]` 的插值点也在曲面上。

## 8. 动画时序（总时长估算）

| 阶段 | 内容 | 时长 |
|------|------|------|
| 0 | 空坐标系展示 | `hold_t` = 0.8 s |
| 1 | 10 根蓝立柱逐根升起 | `N·dt` = 10×0.15 = 1.5 s |
| - | 阶段间停顿 | `hold_t` = 0.8 s |
| 2 | 曲面片淡入 | 0.3 s |
| 2 | 16 条橙母线逐条延展 | `(M_u−1 + M_v−1)·dt` ≈ 16×0.15 = 2.4 s |
| - | 终态停留 | 1.5 s |
| **合计** | | **≈ 7.3 s** |

> 录制 mp4 帧数 ≈ `7.3 × 30 ≈ 220 帧`。

## 9. 错误处理与边界

- 参数非法（`N<3`, `a<=0`, `c<0`）：`nest_geometry` 抛 `error('NEST:param', ...)`，附中文提示。
- `VideoWriter` 不可用（无编解码器）：捕获警告，降级为只跑实时动画，提示用户。
- 母线条数过少（`M<2`）：自动夹到 `M=2`。
- 域 `u,v∈[−1,1]` 严格闭区间，避免浮点抖动导致母线端点偏出曲面（中点测试容差覆盖）。
