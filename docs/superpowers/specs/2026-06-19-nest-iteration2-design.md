## 2. 数学原理

### 2.1 双曲抛物面（Hyperbolic Paraboloid）

**标准方程**（含偏移）：

```
z = c · (x²/ahp² - y²/bhp²) + z_offset
其中 ahp=80, bhp=72, c=8, z_offset=5
```

**直纹面特性**：过曲面上每一点有且仅有两条直线完全落在曲面上，分别属于两族直母线（u族和v族）。

**因式分解**：

```
z = c · (x/a - y/b) · (x/a + y/b)
```

令 `u = x/a - y/b`, `v = x/a + y/b`，则 `z = c · u · v`。

- **u族**（v = const）：x/a + y/b = 常数 → xy平面投影斜率 = -b/a
- **v族**（u = const）：x/a - y/b = 常数 → xy平面投影斜率 = +b/a

两族夹角 = 2·arctan(b/a)。当 b/a ≈ 0.89 时，夹角 ≈ 84°，接近正交，形成规则的类直角编织网格。

### 2.2 鸟巢屋顶的几何构造

鸟巢屋顶实质是双曲抛物面被内外两个椭圆截取后形成的环形区域：

| 参数 | 实际尺寸 | 缩放后（动画用） |
|------|---------|-----------------|
| 外壳椭圆长半轴 | 332.3 m | **a_out = 60** |
| 外壳椭圆短半轴 | 296.4 m | **b_out = 53.5** |
| 开口椭圆长半轴 | ~190.3 m | **a_in = 34.4** |
| 开口椭圆短半轴 | ~124.1 m | **b_in = 22.4** |
| HP曲面a参数 | — | **ahp = 80**（>a_out，曲率更缓） |
| HP曲面b参数 | — | **bhp = 72**（>b_out） |
| 翘起量 | — | **c = 18** |
| 垂直偏移 | — | **z_offset = 23.2** |
| u/v参数范围 | — | **[-1.06, 1.06]** |
| 外层边界z范围 | — | ≈ 13.3 ~ 33.3（高度差≈20，课堂演示清晰） |
| 相机视角 | — | **az = -20°, el = 25°** |

### 2.3 构造流程

```
1. 定义 HP 曲面 z = c·(x²/ahp² - y²/bhp²)
2. 生成 u族直母线（50条）：x/ahp - y/bhp = u，u ∈ [umin, umax]
3. 生成 v族直母线（50条）：x/ahp + y/bhp = v，v ∈ [vmin, vmax]
4. 每条直母线与外椭圆 (x/a_out)²+(y/b_out)²=1 求交
5. 每条直母线与内椭圆 (x/a_in)²+(y/b_in)²=1 求交
6. 立柱位于交点处：从 z=0 到 z=hpZ(x,y)
7. 屋顶线：沿直母线连接相邻交点，**线段端点必须落在立柱顶端**
   - 外→内（跨越屋面膜区域）
   - 外→外（直母线未经过内椭圆时）
   - 内→外（对称的另一侧）
   - ⚠ 屋顶线是有限线段，端点 = 立柱顶端，而非无限直线
8. 仅保留内外椭圆之间的线段（中间开口区域无屋顶）
```

## 3. 动画分镜

### 3.1 阶段0 · 空坐标系

| 属性 | 值 |
|------|-----|
| 持续时间 | ~1 s |
| 元素 | X/Y/Z 三轴（带标注）+ 地面参考网格 |
| 相机 | az = -35°, el = 30°（固定不变） |
| 背景色 | 白色或浅灰 |

**目的**：建立空间参照系，让学生预期后续构件落点。

### 3.2 过渡0→1 · 立柱逐根生长

| 属性 | 值 |
|------|-----|
| 持续时间 | ~2-3 s |
| 生长方式 | 每根立柱从 z=0 线性 interpolate 到目标高度 |
| 生长速度 | ~0.03 s/根 |
| 生长顺序 | 按角度排列顺序（反时针，沿外椭圆 → 内椭圆） |
| 颜色 | **蓝色 #0072BD** |

**实现**：逐帧更新，每帧新增 N 根立柱（N 由总时间和立柱数自动计算），每根立柱在帧内从 0→目标高度动画。

### 3.3 阶段1 · 立柱完成

| 属性 | 值 |
|------|-----|
| 持续时间 | ~2 s |
| 内容 | 内外椭圆上所有立柱竖立完成 |
| 立柱顶端 | 恰好落在 HP 曲面上，高度各异 |

### 3.4 过渡1→2 · 屋顶编织线逐条铺出

| 属性 | 值 |
|------|-----|
| 持续时间 | ~3-4 s |
| 铺出方式 | 每条直母线从一端延伸到另一端 |
| 铺出速度 | ~0.05 s/条 |
| 铺出顺序 | 先 u 族全部（50条），再 v 族全部（50条） |
| 颜色 | **橙色 #EDB120** |

### 3.5 阶段2 · 屋顶编织网完成（最终定格）

| 属性 | 值 |
|------|-----|
| 持续时间 | ~3 s |
| 内容 | 完整的两族直母线交叉编织网 |
| 无 | 半透明曲面、地面黑圈（纯直线构造） |
| 立柱可见 | 是（蓝色，与橙色编织线共存） |

## 10. 附录：关键数学公式

### 10.1 直线与椭圆求交

给定直线 `y = mx + d` 和椭圆 `(x/a)² + (y/b)² = 1`：

```
代入： (x/a)² + ((mx+d)/b)² = 1
整理： (1/a² + m²/b²)·x² + (2md/b²)·x + (d²/b² - 1) = 0
求解二次方程，得到 0/1/2 个交点。
```

### 10.2 u/v族直母线端点

对于 u族第 i 条（u = uᵢ），直线方程为 `y = (bhp/ahp) · (x - ahp·uᵢ)`：

1. 与该直线求交外椭圆 → 最多2个交点
2. 与该直线求交内椭圆 → 最多2个交点
3. 沿直线排序所有交点
4. 连接相邻交点（跳过跨过开口的线段）

v族同理（斜率 = -bhp/ahp）。
